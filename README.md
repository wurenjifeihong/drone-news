# 无人机影像标注工具

## 快速开始
双击 **标注工具.exe** 即可启动（无需安装任何软件）。
或双击 **标注工具.html** 在浏览器中打开。

## 文件说明
| 文件 | 用途 |
|------|------|
| 标注工具.html | 主程序（标注、导出等全部功能） |
| 标注工具.exe  | Windows 桌面启动器（内嵌HTML，双击即用） |
| build_exe.ps1  | 重新生成 EXE 的脚本 |

## 部署到新电脑
1. 复制整个 `无人机` 文件夹到新电脑
2. 双击 `标注工具.exe` 即可使用
3. 如 EXE 无法运行，双击 `标注工具.html` 用浏览器打开

## 重新生成 EXE
在 PowerShell 中运行：
```
.\build_exe.ps1
```
需要 .NET Framework 4.x（Windows 10/11 自带）。

## 功能
- 手绘多边形标注 / 魔术棒自动矩形框选
- 图层管理：多选、批量改色、显隐切换
- 线宽、容差可调
- 导出 JSON（坐标数据）/ PNG（标注图片）
- Catmull-Rom 平滑曲线 / 标准直线线框
- 无重叠框选算法

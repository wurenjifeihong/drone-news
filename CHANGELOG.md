# 更新日志

## v1.0.0 (2026-06-01)
### 核心功能
- 手绘多边形标注 + Catmull-Rom 平滑曲线
- 自动框选：魔术棒颜色识别 + 矩形范围框选
- 吸管取色 + 放大镜预览
- 左侧图层面板：多选(Ctrl+点击)、批量改色、显隐切换

### 数据管理
- 按图片×类别双层存储标注（空鼓检查 / 裂缝检查）
- 切换图片/类别自动保存，未保存提示
- 导出 JSON（含坐标、颜色、类别）+ PNG 无损图片

### 界面
- 一级导航：空鼓检查 / 裂缝检查
- 线宽 1~10 可调，线框/名称独立显隐
- 容差滑块 0~100
- 颜色选择器（12 预设 + 自定义）
- 重叠检测（零容忍）

### 工程
- 标注工具.html — 主程序（单文件）
- 标注工具.exe — .NET 桌面启动器
- build_exe.ps1 — 一键重新编译
- README.md / AGENTS.md — 文档
